Query multiple (medical) search engines

This tool can be used to compare search results and their quality. Search results will be displayed in several new tabs. Make sure that your browser / ad blocker allows this. I suggest that you open this page in a new browser window before starting a new search, so that you can close all tabs at once by closing the window.

The following search engines are currently queried:

  • google.com
  • tripdatabase.com
  • Medical Search Enginge, Google CSE Baseline (A Google Custom Search Engine developed by Matthias Samwald)
  • FindMeEvidence
  • HON search (general)
  • HON search (for medical professionals)
  • HON Khresmoi prototype (production version)
  • PubMed
  • PubMed Clinical Queries
  • Bing

Need ideas for queries? Try some queries from the famous 'Googling for Diagnosis' paper (Tang et al. 2006) or the Khresmoi list of 50 example queries for physicians (Excel file, requires login).
